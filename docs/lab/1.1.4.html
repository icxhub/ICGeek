Tcl Seminar: 学习Tcl (Day 4)

今天来看看 Tcl 对几个数据库的接口，Tcl 的扩展能力很强，所以基本上编写一个数据库接口对于熟悉 C 的程序员来说并不复杂。请尝试熟悉使用连接以下数据库：

  1. Mysql
  2. Sqlite3
  3. neo4j

推荐随便翻翻 SQL 的知识，比如 《Sql入门经典》，虽然现在 No-SQL 也很流行，但是并不见得 SQL 没有优势。
